break与continue一般只出现在循环语句中

break 语句
break ： 跳出for或while循环，执行到break语句，当前循环彻底结束。
break : 只能控制离他最近的循环
break : 循环语句是可以有else语句，当执行到break语句时，会导致循环 彻底结束，else语句也不再执行。

continue 语句

continue : 当循环执行到continue语句时，跳出本次循环，不影响下一次循环
continue ： 只能控制离他最近的循环
